# Paper Tables

This folder contains rounded CSV tables prepared for paper writing and GitHub display.

## Files

- `table_1_base_paper_comparison_rounded.csv`
  - Compares the base paper result with the final ToN-IoT-rareclass-IDS model.

- `table_2_ablation_results_rounded.csv`
  - Reports ablation results across XGBoost, CatBoost, LightGBM, correlation pruning, and MITM sampling ratios.

- `table_3_per_class_report_rounded.csv`
  - Reports per-class precision, recall, F1-score, and support for the final model.

Full raw result files are stored in the `results/` directory.
